# AW Website Redesign

Personal website for Alistair Webster — coach and thinking partner.

## Tech Stack
- Next.js 14
- Tailwind CSS
- Vercel hosting

## Key Files
| File | Purpose |
|------|---------|
| `app/page.tsx` | Home page |
| `app/about/page.tsx` | About page |
| `app/working/page.tsx` | Working Together page |
| `app/globals.css` | All styling |
| `components/Navigation.tsx` | Navigation |
| `components/Footer.tsx` | Footer |

## Commands
```bash
npm run dev    # Local dev server
npm run build  # Production build
```
